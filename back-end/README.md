# wad-homework-4-bf

## Pre-requisites
Node.js \& NPM

## Clone the repository
```
git clone https://github.com/armeig/wad-homework-4-bf.git
```

## Navigate to the project directory
```
cd <project name>
```

## Install dependencies
```
npm install
```

## Run the project
```
npm start
```

or 
```
npm server.js
```
